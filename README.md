# NotasMIDI

## About

This library is being developed by @piruetasxyz, the art studio by @montoyamoraga.

## Dependencies

None

## Formatting

### Solfeo

- `DO`
- `RE`
- `MI`
- `FA`
- `SO`
- `LA`
- `SI`

### Letras

- `A`
- `B`
- `C`
- `D`
- `E`
- `F`
- `G`

### Accidentals

- `^x`: x sharp
- `_x`: x flat
- `=x`: x natural

## Functions

| name           | input               | output              |
| :------------- | :------------------ | :------------------ |
| solfeoToNumero | note in solfeo      | note in MIDI number |
| numeroToSolfeo | note in MIDI number | note in solfeo      |
